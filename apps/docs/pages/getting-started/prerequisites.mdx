# Prerequisites

Before installing Vortex, ensure your server meets the following requirements.

## System requirements

| Requirement | Minimum | Recommended |
|---|---|---|
| OS | Ubuntu 22.04 / Debian 12 / any Linux with Docker | Ubuntu 22.04+ |
| RAM | 4 GB | 8 GB |
| CPU | 2 cores | 4 cores |
| Disk | 10 GB | 20 GB+ (more for large knowledge base uploads) |
| Docker | 24.0+ with Compose v2 | latest |
| Python | 3.12+ (manual install only) | 3.12 |
| Node.js | 20+ (manual install only) | 20 LTS |

## External services

### Database (required)

PostgreSQL 15+ with the **pgvector extension** enabled. This is required for knowledge base functionality (RAG).

import { Callout } from 'nextra/components'

<Callout type="info">
The Docker Compose path uses `pgvector/pgvector:pg17` which includes the extension pre-installed. For manual installs, see [pgvector installation](https://github.com/pgvector/pgvector#installation).
</Callout>

### LLM API key (required — at least one)

Vortex does not bundle a model. You must supply at least one API key:

| Provider | Environment variable | Get a key |
|---|---|---|
| Anthropic (Claude) | `ANTHROPIC_API_KEY` | [console.anthropic.com](https://console.anthropic.com) |
| OpenAI / compatible | `OPENAI_API_KEY` | [platform.openai.com](https://platform.openai.com) |
| Google Gemini | `GEMINI_API_KEY` | [aistudio.google.com](https://aistudio.google.com) |

### Embedding provider (required for knowledge bases)

If you plan to use knowledge bases, you need an embedding provider:

| Provider | Key | Notes |
|---|---|---|
| Voyage AI | `VOYAGE_API_KEY` | **Recommended.** `voyage-4-lite` model — 200M tokens/month free. |
| OpenAI | `OPENAI_API_KEY` | Falls back to OpenAI embeddings if `VOYAGE_API_KEY` is not set. |

<Callout type="warning">
An Anthropic API key does **not** provide embeddings. Set `VOYAGE_API_KEY` or `OPENAI_API_KEY` separately for knowledge base functionality.
</Callout>

### SMTP (required for selfhosted/saas modes)

Email is used for user verification and password reset. Any SMTP provider works (Resend, SendGrid, Mailgun, Gmail SMTP, etc.).

Not required in `dev` mode.

## Optional

- **Redis 7+** — not required today; Vortex runs ingest in-process. Included in Docker Compose for future use.
- **Firecrawl / Jina API key** — for the `fetch_webpage` tool (web scraping). The tool works without these via a fallback chain.
